# JSX解析原理

JSX语法糖的原理是什么，先来了解React的一个函数。

# creatElement

React.createElement(
  type,
  [props],
  [...children]
)

第一个参数必填，是html标签名
第二个参数就是React中组件的Props
第三个组件是子组件

  • 下面看一个jsx转js的例子

jsx

render() {
  const list = this.props.data
  return (
    <ul>
      {list.map((item,index)=>{
        return <li key={index}>{item}</li>
        })
      } 
  )
}

js

function render(){
  const list = this.props.data
  return React.createElement(
    "ul",
    null,
    list.map((item)=>{
      return React.createElement(
        "li",
        {key: index},
        item
      )
    })
  )
}
